# BrainDump

## Overview
BrainDump is a structured knowledge management system designed for organizing technical notes and documentation. It provides a minimalist interface centered on content readability and efficient information browsing.

## Core Features
- Multilingual Support: Full integration for English and Vietnamese content at both the interface and data layers.
- Hierarchical Organization: Automatic generation of navigation trees based on the physical folder structure.
- Clean Readability: Optimized typography and markdown rendering for a distraction-free reading experience.
- Seamless Navigation: Smooth transitions between categories and locales to ensure a continuous browsing flow.

## Technical Stack
- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- Next-intl
- Unified / Remark / Rehype

## Installation and Execution

### Prerequisites
- Node.js (v18 or higher)

### Setup
Clone the repository and install dependencies:
```bash
git clone https://github.com/huynhanx03/BrainDump.git
cd BrainDump
npm install
```

### Development
Start the application locally:
```bash
npm run dev
```
Access the interface at http://localhost:3000.
