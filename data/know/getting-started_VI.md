# Bắt đầu với BrainDump

Chào mừng bạn đến với cơ sở kiến thức cá nhân của bạn. Đây là nơi bạn có thể lưu trữ các ghi chú, đoạn mã và suy nghĩ của mình.

## Cách thức hoạt động

1. Tất cả các ghi chú được lưu trữ dưới dạng tệp Markdown trong thư mục `/data`.
2. Trang web sẽ tự động đọc cấu trúc thư mục và tạo thanh bên.
3. Bạn có thể có nhiều phiên bản của một ghi chú (ví dụ: `_EN.md` và `_VI.md`).

## Ví dụ về mã

Đây là cách bạn có thể viết một số mã Go:

```go
package main

import "fmt"

func main() {
    fmt.Println("Chào mừng bạn đến với BrainDump!")
}
```
