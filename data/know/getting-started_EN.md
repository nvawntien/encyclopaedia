# Getting Started with BrainDump

Welcome to your personal knowledge base. This is a place where you can store your notes, snippets, and thoughts.

## How it works

1. All notes are stored as Markdown files in the `/data` folder.
2. The website automatically reads the folder structure and generates a sidebar.
3. You can have multiple versions of a note (e.g., `_EN.md` and `_VI.md`).

## Code Example

Here is how you might write some Go code:

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, BrainDump!")
}
```
